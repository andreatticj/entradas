Explicación de la Aplicación WebChecker

WebChecker permite monitorear una página web en busca de una palabra clave específica y notifica al usuario cuando la encuentra.

Funcionamiento

  1. El usuario ingresa una URL y una palabra clave en la interfaz de la aplicación.
  2. Al presionar el botón Play, se inicia un Worker en segundo plano.
  3. Este Worker usa la librería Jsoup para conectarse a la página web y buscar la palabra clave.
  4. Si la palabra clave es encontrada, se genera una notificación.
  5. El usuario puede detener la búsqueda presionando Stop o desde la notificación.

Tecnologías Utilizadas

  • Android WorkManager: Para tareas en segundo plano.
  • Jsoup: Para la extracción y análisis del contenido web.
  • SharedPreferences: Para almacenar datos ingresados por el usuario.
  • Notificaciones: Para alertar cuando se encuentra la palabra clave.

Interfaz de Usuario

  • Campo para ingresar la URL.
  • Campo para ingresar la palabra clave.
  • Botón Play para iniciar la búsqueda.
  • Botón Stop para detener la búsqueda.

Notificaciones

Cuando se encuentra la palabra clave, se genera una notificación con la opción de detener la búsqueda.

Código Relacionado

  • MainActivity.kt: Configura la UI y gestiona WorkManager.
  • WebCheckerWorker.kt: Ejecuta la tarea de fondo.
  • NotificationReceiver.kt: Permite detener el Worker desde la notificación.
